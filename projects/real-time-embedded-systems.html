Embedded Sentry System

Abstract

This project implements an "Embedded Sentry" system using gesture recognition for secure access control. The system utilizes a single accelerometer/gyroscope to record and authenticate hand movement sequences, providing a novel approach to protecting assets and data.

Methodology

The project methodology included the following key steps:

  • Design and implementation of gesture recording functionality
  • Development of gesture mapping and comparison algorithms
  • Integration of visual feedback system using LEDs
  • Implementation of "Record Key" feature for sequence storage
  • Calibration and testing of the gesture recognition system

Implementation Details

The Embedded Sentry system was developed using the following components and techniques:

  • Microcontroller: STM32F49I Discovery Board with integrated accelerometer/gyroscope
  • Development Environment: PlatformIO IDE
  • Programming Language: C++
  • Key Functions:
    • record_gesture_data(): Captures motion data during gesture input
    • map_gesture_data(): Converts raw sensor data to directional values
    • compare_gesture(): Matches input gestures against stored sequences
    • calibrate_sensor(): Ensures accuracy of the motion sensor
  • User Interface: Push button for initiating gesture recording and LED indicators for system feedback

Results and Discussion

The Embedded Sentry project achieved the following outcomes:

  • Successful implementation of gesture-based unlock mechanism
  • Secure storage of gesture sequences on the microcontroller
  • Visual feedback system for user interaction and status indication
  • Robust gesture recognition with tolerance for minor variations

The system demonstrated high accuracy in gesture recognition, providing a secure and innovative approach to access control.

Conclusion

The Embedded Sentry project showcases the potential of gesture-based security in embedded systems. By leveraging motion sensors and advanced algorithms, it offers a unique solution for protecting assets and data, opening new possibilities in the field of embedded security systems.
